# Contributing to ArgonautDiceBot

Thank you for your interest in contributing!

This project is currently in early alpha (v0.1-alpha). The goal of this phase is stability testing and feedback.

## How to Report Bugs

Please open an Issue and include:

- What command you used
- What you expected to happen
- What actually happened
- Any console errors
- .NET version used

Clear reports help fix problems faster.

## Feature Requests

Feature requests are welcome, but major structural changes will be considered for future versions (v0.2+).

## Code Contributions

If you would like to submit code:

- Keep changes focused and small
- Follow existing formatting style
- Explain what your change does and why

Thank you for helping improve the bot!
